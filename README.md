## Experiência em Destaque

<details>
<summary><strong>VTEX — Integração completa de pedidos e PBM</strong></summary>
<br/>
Integração com a API da VTEX para captura de pedidos completos, incluindo fluxos de PBM e compras comuns. Dados obtidos via VTEX IO com consolidação em SQL Server interno para controle e análise.
</details>

<details>
<summary><strong>Conde Negócios — Dashboard de posicionamento digital</strong></summary>
<br/>
Dashboard desenvolvido com Bootstrap + ECharts, com segmentação por canal (site / app Android / app iOS), vendas por hora e previsibilidade do dia. Visão real de posicionamento digital para a liderança.
</details>

<details>
<summary><strong>iFood — Coleta centralizada de 260+ lojas</strong></summary>
<br/>
Sistema de coleta centralizada de pedidos de mais de 260 lojas iFood, com armazenamento em base interna para controle, acompanhamento e tomada de decisão.
</details>

<details>
<summary><strong>Marketplaces — Mercado Livre via AnyMarket</strong></summary>
<br/>
Obtenção e liberação de documentos fiscais (NF/XML) do Mercado Livre via AnyMarket, com envio para o ERP Procfit. Fluxo fiscal integrado e automatizado.
</details>

<details>
<summary><strong>TikTok Shop — Implantação do fluxo fiscal</strong></summary>
<br/>
Desenho e implantação do fluxo fiscal completo para o TikTok Shop, com integração de notas via ERP e retorno de informações fiscais para a plataforma.
</details>

<details>
<summary><strong>LOCUS — Rastreio CD → Lojas</strong></summary>
<br/>
Rastreio de volumes e romaneios no fluxo CD → Lojas, validações finais de processo e apoio a rotinas de estoque e movimentações.
</details>

<details>
<summary><strong>Automações — n8n + WhatsApp + Python + Node.js</strong></summary>
<br/>
Automações com n8n integrado ao WhatsApp. Python para processamento de XML de NF-e. Node.js para integrações e rotinas internas diversas.
</details>
